# Place your case images here

## Folder structure:
```
cases/
├── case-001/
│   ├── extraoral-frontal-pre.webp
│   ├── extraoral-frontal-post.webp
│   ├── extraoral-profile-pre.webp
│   ├── extraoral-profile-post.webp
│   ├── intraoral-upper-pre.webp
│   ├── intraoral-upper-post.webp
│   └── thumbnail.webp
├── case-002/
│   └── ...
```

## Naming convention:
- `extraoral-frontal-pre.webp` - Front face before
- `extraoral-frontal-post.webp` - Front face after
- `extraoral-profile-pre.webp` - Side profile before
- `intraoral-upper-pre.webp` - Upper arch before
- `opg-pre.webp` - OPG X-ray before
- `thumbnail.webp` - Card thumbnail (600x400px)
