## Folder structure:
```
cases/
├── case-001/
│   ├── extraoral-frontal-pre.jpg
│   ├── extraoral-frontal-post.jpg
│   ├── extraoral-profile-pre.jpg
│   ├── extraoral-profile-post.jpg
│   ├── intraoral-upper-pre.jpg
│   ├── intraoral-upper-post.jpg
│   └── thumbnail.jpg
├── case-002/
│   └── ...
```

## Naming convention:
- `extraoral-frontal-pre.jpg` - Front face before
- `extraoral-frontal-post.jpg` - Front face after
- `extraoral-profile-pre.jpg` - Side profile before
- `intraoral-upper-pre.jpg` - Upper arch before
- `opg-pre.jpg` - OPG X-ray before
- `thumbnail.jpg` - Card thumbnail (600x400px)
